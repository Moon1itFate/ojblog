# 周复盘模板

周复盘用于把 OJ Tracker 的原始记录整理成可执行的训练计划。它不追求堆数字，重点是找出下一周要修什么。

## 快速生成

```bash
corepack pnpm new:post -- --type review --title "第 N 周刷题复盘" --slug weekly-review-n
```

生成后的文章会放在 `src/content/blog/review`。

## 推荐结构

```md
## 本周概览

- 主要平台：
- 训练主题：
- 完成情况：

## 数据记录

| 指标 | 数值 | 备注 |
| --- | --- | --- |
| 提交数 |  |  |
| 通过数 |  |  |
| 参与比赛 |  |  |
| 重点题型 |  |  |

## 薄弱题型

- 题型：
- 表现：
- 原因：
- 补救计划：

## 错因模式

- 待补充

## 值得写成题解的题

- 待补充

## 下周计划

- 待补充
```

## 写作标准

- 每周至少留下一个“具体薄弱点”。
- 每周至少挑一题进入题解候选。
- 下周计划要能执行，不写笼统口号。
- 如果有 AI 总结，先把 AI 输出改成人能读的复盘，再发布。
