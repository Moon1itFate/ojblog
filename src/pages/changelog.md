## 2026-06-01

### 个人博客骨架

- 新增 [Start Here](/start) 入口页，让新读者能快速找到内容。
- 新增 [Now](/now) 页面，记录当前写作、刷题和项目维护重点。
- 新增 [Projects](/projects) 页面，整理站点和 OJ Tracker 的长期计划。
- 增强 [About](/about) 页面，补充读者入口、RSS 和评论说明。

### 评论与互动

- 接入 Giscus 评论配置，等待 GitHub Discussions 信息填入后即可启用。
- 新增 [留言板](/guestbook)，用于站点级交流和反馈。
- 新增评论配置文档 `docs/comment-setup.md`。

### 写作与维护

- 新增文章发布流程文档 `docs/writing-workflow.md`。
- 新增 `pnpm new:post` 文章生成器。
- 新增周复盘模板，方便固定节奏沉淀刷题记录。

### OJ Tracker

- 增强牛客主页刷题记录同步。
- 增加洛谷记录同步入口。
- 新增 [刷题追踪说明](/tracker-guide)，解释支持平台、同步限制和未来方向。

## 维护说明

这个页面只记录对读者和维护者有意义的变化。更细的提交历史可以查看 GitHub 仓库。
